# NewLine 2026

A minimalist landing page for Hackerspace Gent's NewLine event.

## Event Details

**NewLine** is a show and tell for hackers, makers and tinkerers of all shapes and kinds.

- **Dates:** 29-30-31 May 2026
- **Website:** newline.gent
- **Organized by:** Hackerspace Gent

## Design

This website features a minimalist design inspired by the event poster, with:

- Beautiful gradient background transitioning from pink to coral to orange/yellow
- Brother typeface (bold) for strong, impactful typography
- Clean, centered layout with focus on essential information
- Fully responsive design that works on all devices
- "Less is more" philosophy - removing unnecessary elements

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and gradient effects
- `fonts/` - Brother typeface files
- `img/` - Event images and icons
  - `poster2026c.png` - Event poster
  - `header.png` - Header image
  - `icons/` - Various event-related icons

## Development

Simply open `index.html` in a web browser to view the site. No build process required.

## Typography

The site uses the Brother typeface, a bold display font that gives the event its distinctive character.
